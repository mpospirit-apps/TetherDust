# 0.4.2 — 2026-06-22

**Deferred fast-follow #2 (Monitoring): chat session transcript.** From the 0.4.0
API-only + SPA migration's deferred-items list.

## Upgrade notes for admins
- No DB migrations and no new env vars. Rebuild the `backend` + `frontend` images
  (`docker compose up --build`) to pick up the changes.

## Added
- **Chat session transcript (Monitoring → Sessions).** Session rows are now
  clickable and open a **detail page** showing the full message-by-message
  transcript — role-coloured user/assistant/system messages (assistant rendered
  as Markdown), per-message `@`-sources, `/`-prompts, and MCP tools used, plus
  timestamps and session metadata. Backed by a new staff endpoint,
  `GET /api/v1/admin/sessions/{id}/`. (Restores the legacy `session_detail_view`,
  which wasn't ported when Sessions shipped read-only in 0.4.0.)
