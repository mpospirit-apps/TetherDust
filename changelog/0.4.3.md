# 0.4.3 — 2026-06-22

**Deferred fast-follow #3 (Monitoring): chart-generation run detail.** From the
0.4.0 API-only + SPA migration's deferred-items list.

## Upgrade notes for admins
- No DB migrations and no new env vars. Rebuild the `backend` + `frontend` images
  (`docker compose up --build`) to pick up the changes.

## Added
- **Chart-generation run detail (Monitoring → Chart Generation).** Rows in the
  generation history are now clickable and open a **detail page** showing the
  full run: status, dashboard, charts created, duration, user, agent, start/finish
  times, the **source databases & documentation** used, any error, and the
  **prompt sent to the agent** plus the **raw agent output**. Backed by the
  existing read-only viewset's `retrieve` route (`GET /api/v1/admin/chartgen-logs/{id}/`)
  with a richer detail serializer — the list stays lean (the prompt/output text is
  detail-only). (Restores the legacy `chartgen_log_detail_view`, which wasn't
  ported when the chart-generation log shipped as a read-only list in 0.4.0.)
