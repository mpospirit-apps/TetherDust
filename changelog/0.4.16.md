# 0.4.16 — 2026-06-27

## Upgrade notes for admins
- No migrations, no new env vars.

## Added
- CI now enforces that every PR to `main` must bump `VERSION` to a value strictly
  greater than the current `main` version and include a matching `changelog/<version>.md`.
  PRs that skip either step will fail the **Version Bump Check** status check.
