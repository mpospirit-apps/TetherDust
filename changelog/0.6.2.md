# 0.6.2 — 2026-07-11

## Upgrade notes for admins
- Frontend-only change. Rebuild and restart the frontend: `docker compose build frontend && docker compose up -d frontend`.

## Changed
- Removed elevation/drop shadows across the design system (cards, modals,
  dropdowns, buttons, chat bubbles, tether panels/tooltips). Rim-light bevel
  highlights, focus rings, selection rings, and small colored glows (fairy
  dust, status dots, hover pulses) are unaffected.
