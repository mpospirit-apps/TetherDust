# 0.6.1 — 2026-07-10

## Upgrade notes for admins
- Frontend-only change. Rebuild and restart the frontend: `docker compose build frontend && docker compose up -d frontend`.

## Added
- **Getting Started page** (Console → Getting Started): a tabbed onboarding
  guide — Welcome, Foundations, Analytics, Role Management, Advanced — that
  walks admins through setting up TetherDust end to end (agent → codebase →
  codebase docs → database → database docs → tethers, then reports/dashboards,
  roles/users, and prompts/MCP servers).
  - Each step shows a live "done/not done" status pulled from existing data
    (active agent, registered codebases/databases, generated doc libraries,
    tethers, reports, dashboards, roles, assigned users, prompts, custom MCP
    servers) with a direct link to complete it.
  - Each step includes an animated SVG diagram illustrating how that piece
    fits into the overall system.
