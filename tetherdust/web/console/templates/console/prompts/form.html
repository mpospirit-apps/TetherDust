Prompts

What is a Prompt?

A prompt is a reusable message template the AI agent can request from the MCP server. Prompts contain static text used to inject structured context into a conversation.

Name & Display Name

The name is the machine-readable identifier used in the MCP protocol (e.g. summarise_table). The display name is shown in the admin UI.

Prompt Content

The message template returned to the agent. The content is sent as-is to the AI agent when this prompt is invoked.

Active vs Inactive

Inactive prompts are hidden from agents even if the parent MCP server is active. Use this to disable a prompt temporarily without deleting it.

Tip: Keep prompt content focused and concise. Prompts work best when they provide precise instructions for a single, well-defined task.
