Doc Generation Logs

What are Doc Generation Logs?

Every time documentation is generated using the AI agent, a log entry is created with timing, status, and error details. This provides an audit trail and helps diagnose issues when generation fails or produces incomplete results.

Status

  • SUCCESS — Documentation was generated without errors.
  • PARTIAL — Documentation was generated but some databases were unavailable. The file contains placeholder text for skipped databases.
  • FAILED — Generation failed entirely. Check the error message for details.
  • RUNNING — Generation is currently in progress.
Tip: If you see frequent PARTIAL results, check the connectivity of your database connections on the Databases page.
