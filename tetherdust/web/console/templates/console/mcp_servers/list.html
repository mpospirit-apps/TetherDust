MCP Servers

What are MCP Servers?

MCP (Model Context Protocol) servers expose tools to AI agents. Each server groups related tools together. When a user sends a query, the agent calls tools from the configured MCP servers to gather information and compose a response.

Built-in Server

TetherDust ships with a built-in MCP server that provides core database querying and documentation tools. This server is always active and cannot be edited or deleted. You can still manage (add, edit, enable, disable) the tools within it.

Adding MCP Servers

You can add additional MCP servers to extend the tools available to agents. Each server can have its own connection settings and set of tools.

Active vs Inactive

Deactivating an MCP server makes all its tools unavailable to agents, regardless of individual tool settings. This is useful for temporarily removing a set of capabilities without deleting configurations.

Tip: Click on an MCP server to view and manage its tools. You can add, edit, enable, and disable individual tools from the server detail page.
