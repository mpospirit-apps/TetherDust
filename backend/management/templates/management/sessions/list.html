Chat Sessions

What are Chat Sessions?

A chat session is a single conversation thread between a user and the AI agent. Sessions are created automatically when a user opens the chat interface and sends their first message. Each session preserves the full message history for that conversation.

Filtering

By default, only sessions that contain at least one message are shown. Click All to include empty sessions — these are sessions that were opened but abandoned before any messages were sent.

Viewing a Session

Click any row to open the session detail view. This shows the full conversation transcript, including both user messages and agent responses, along with any tool calls the agent made during the session.

Session Retention

Sessions and their messages are stored indefinitely. Deleting a user account will remove their sessions. There is no automatic expiry or cleanup — manage storage at the database level if needed.

Tip: Review sessions for users who report incorrect answers. The full transcript — including tool calls — shows exactly what the agent queried and why it produced a given response.
