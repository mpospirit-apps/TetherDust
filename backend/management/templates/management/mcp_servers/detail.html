MCP Server Detail

Tools

Tools are functions the AI agent can call during a conversation. The built-in tools are a fixed mirror of the code that ships with TetherDust, so they are read-only — they cannot be added, edited, or deleted here.

Click View next to a tool to see what the agent actually receives: its description, input schema, and handler code (all derived live from the Python module). To grant or revoke tools for specific users, use allowed tools on a Role.

Prompts

Prompts are reusable message templates the agent can request from this server. They can contain static text or dynamic content driven by arguments, and are used to inject structured context into a conversation. Prompts can be added, edited, and enabled/disabled from this page.

Built-in Server

The built-in TetherDust server and its tools cannot be edited or deleted. To change a tool's behaviour, description, or schema, edit its Python module in mcp_server/tools/ and restart the MCP server.

Tip: The agent relies on each tool's docstring (shown on the View page) to decide when to call it — that is the real source of truth, not anything stored in the management.
